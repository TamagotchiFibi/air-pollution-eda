# Анализ качества атмосферного воздуха в мировом масштабе

EDA-проект на основе датасета с Kaggle (2023).

**Стек:** Python, pandas, SQLite, matplotlib, seaborn

**Что внутри:**
- Очистка данных и работа с пропусками
- SQL + pandas для анализа
- Визуализация топ-10 загрязнённых стран
- Определение основных загрязнителей

**Датасет:** [Kaggle](https://www.kaggle.com/datasets/hasibalmuzdadid/global-air-pollution-dataset)
